# Cockpit Frontend

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint --fix
```

### Runs e2e tests in UI

```
npm run test:e2e
```

### Runs e2e tests without UI
```
npm run test:e2e:headless
```
